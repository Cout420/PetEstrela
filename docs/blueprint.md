# **App Name**: Pet Estrela Crematório

## Core Features:

- Home Page Carousel: Display a carousel with images and texts promoting the pet crematorium services.
- Digital Memorial: Allow users to view digital memorials of pets with photos and memorial text.
- Service Plans: Display different service plans with detailed information about each.
- Contact Information: Provide complete contact information for users to get in touch.

## Style Guidelines:

- Background color: Very light gray (#FBFBFB) for a clean, calming backdrop.
- Primary color: Pet Estrela Blue (#117EA2) for key brand elements.
- Accent color: Luxury Gold (#E1B15A) for highlights and calls to action, especially on premium plan items.
- Headline Font: 'Playfair Display' serif for elegant headings. Note: currently only Google Fonts are supported.
- Body Font: 'Inter' sans-serif for clean and readable text. Note: currently only Google Fonts are supported.
- Subtle fade-in effects for page elements and smooth transitions.
